Preguntas frecuentes

¿Qué es OrderEat?

OrderEAT es un sistema de gestión para comedores y cafeterías de escuelas y colegios conectado a una app para dispositivos móviles, permitiendo a sus clientes hacer pedidos y gestionar sus cuentas de manera fácil y conveniente.

¿Qué necesito para usar OrderEAT?

Para utilizar OrderEAT se requiere solamente una computadora con conexión a Internet en el punto de venta.

¿Qué costo tiene usar OrderEAT para el colegio?

OrderEAT tiene un costo mensual fijo que varía según el país.

¿Qué costo tiene usar OrderEAT para los apoderados?

La aplicación de OrderEAT no tiene ningún costo para los apoderados de los niños.

¿Cuáles son los beneficios principales para el colegio?

Los beneficios principales son reducción de desperdicios, optimización del tiempo de gestión, aumento de ventas y mejor experiencia para alumnos y padres.
